# THydra

Simple script to install thc-hydra in termux

# Installation

<pre>
<code>
pkg update && pkg upgrade -y && pkg i git -y
git clone https://github.com/grayhatzone/THydra.git
cd Thydra && chmod +x * && ./install.sh
</code>
</pre>
